## Problem Statement

Farmers often rely on traditional knowledge or trial-and-error methods to select crops. This can lead to poor yield, soil degradation, and financial losses. Manual assessment of soil and climate suitability is difficult and time-consuming. A data-driven crop prediction system provides a faster, scientific, and scalable solution.
